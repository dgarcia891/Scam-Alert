# Workflow: handoff

1. Context
   - State:
     - Current branch and last commit.
     - Scope of work.

2. Status
   - List:
     - What is done.
     - What is in progress.
     - What is blocked.

3. Next Steps
   - Provide:
     - Immediate next actions.
     - Any open questions.
     - Links or file paths to key artifacts.
