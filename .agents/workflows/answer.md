# Workflow: answer

1. Clarify Question
   - Restate what is being asked.

2. Gather Evidence
   - Use allowed sources (code, docs, logs, official docs) to gather facts.

3. Answer Structure
   - Provide:
     - Direct answer.
     - Supporting reasoning.
     - References to specific files/sections where relevant.

4. Hallucination Controls
   - Apply claim typing and confidence tagging.
   - Prefer “Not found” over guessing.
