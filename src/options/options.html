⚙️ Scam Alert Settings

Scanning Options

When enabled, Scam Alert scans the website you visit and shows a warning if it finds serious concerns.
  • Checks each main page you navigate to (not background tabs).
  • Looks at the website address (the URL) and any enabled reputation checks.
  • Does not read or upload the words on the page itself.
  • Skips internal browser pages and localhost for safety/performance.
  • You can also run a manual scan from the extension popup.
Uses Google’s Safe Browsing service to check whether a website address is known for phishing or malware.
  • Sends the current page URL to Google Safe Browsing (URL reputation check).
  • Requires your own API key to be entered below.
  • Does not upload page text, passwords, or form entries.
Checks the website address against the PhishTank community database.
  • This option is currently not recommended unless you have a working PhishTank setup.
  • May require an API key depending on configuration.
  • Does not read or upload the words on the page itself.
Runs local, offline checks against common scam and phishing patterns in the website address (the URL).
  • Suspicious domain endings (TLDs).
  • Brand impersonation (typosquatting).
  • Hidden/disguised links and URL obfuscation.
  • Too many subdomains or unusual URL structure.
  • Concerning words in the URL like “verify”, “login”, “urgent”, etc.
  • Does not scan the page’s text/content (only the address).
Shows a system notification when Scam Alert finds a high-confidence threat.
  • Notifications appear only for the highest severity results.
  • You can still review details in the extension popup.
Sends a gentle notification when Scam Alert detects that a page is using an unencrypted (HTTP) connection.
  • Combines with the main notification setting above.
  • Triggered only when the address is HTTP and no other warning is present.
  • Does not send page content or form data—only the URL is used.

API Configuration

To enable full protection, please provide your own API keys. These are stored locally on your device.
